### 最長片道切符の旅　###

//○ 変数の定義
//datas : 経路の情報を辞書的に格納
//start : 各経路のスタート側
//end 　: 各経路のゴール側
//distance : 各経路の距離

//node : 現在のノード
visited  : すでに通っている駅
route : 今の経路
t_distance : 合計の距離
max_info : 最大の情報　[0]・・最大距離、[1]・・最大距離の時の経路

○ 流れ
① データを入力(入力が全て反映できるようにwhile文でループ)
② add_line関数
 ・登録されてない経路があれば登録
 ・両方から計算できるように登録(1→4,4→1)
③ find_route関数
 ・最長の経路を探索
 ・格納したデータに対して最長の経路を探索する(all_route関数内で)
④ all_route関数
 ・全ての経路の中で長い経路を更新していく
④ find_route関数の結果を出漁
